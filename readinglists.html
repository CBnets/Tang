Data Statistic Analysis

  • Significance test: For subjective ratings, depending on the design, if samples are correlected, first run a Friedman's test, then run appropriate pairwise Wilcoxon signed rank test. If it is a between subject design and samples are not correlated, use Kruskal Wallis test, then a Mann Whitney as a post-hoc.
